## Function Practice Problems

### 1. Basic function creation
- Write a function `greet(name)` that returns `"Hello, <name>!"`.
- Write a function `square(num)` that returns the square of a number.
- Write a function `isEven(num)` that returns `true` when `num` is even and `false` otherwise.

### 2. Parameters and return values
- Write `sum(a, b, c)` to return the total of three numbers.
- Write `capitalize(word)` to return the word with the first letter uppercase and the rest lowercase.
- Write `getFullName(firstName, lastName)` to combine names with a space.

### 3. Arrays and functions
- Write `doubleArray(values)` that returns a new array with each number doubled.
- Write `findMax(numbers)` that returns the largest number in an array.
- Write `filterShortWords(words)` that returns only words shorter than 5 characters.

### 4. Problem-solving with functions
- Write `fahrenheitToCelsius(f)` to convert Fahrenheit to Celsius.
- Write `countVowels(text)` that returns how many vowels are in a string.
- Write `isPalindrome(str)` that returns `true` if the string reads the same backward and forward.

### 5. Advanced function use
- Write `createMultiplier(factor)` that returns a new function multiplying its input by `factor`.
- Write `randomChoice(items)` to return a random element from an array.
- Write `calculateGrade(score)` to return `"A"`, `"B"`, `"C"`, `"D"`, or `"F"` based on a numeric score.
